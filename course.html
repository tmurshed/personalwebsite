CSC 4190: Computer Network Systems and Applications

Course Description:

With no prior networking knowledge required, this course provides a full overview of computer networking. A top-to-bottom approach is leveraged to understand essential network concepts and protocols, from high-level applications to low-level operational functions, including HTTP, TCP/IP, routing, LAN, wireless, etc. Moreover, you will learn socket programming for data communication.

CSC 4320: Systems Administration

Course Description:

This course introduces students to Linux operations and system administration through a unified virtual machine installation. Students will gain experience working with differing hardware configurations while testing interoperability between Linux and Windows. They will learn about and practice Linux administration with helpful in-text features such as learning objectives, key terms, and self-assessment through review questions, hands-on activities, and more.
